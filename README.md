## About Me

I'm a developer with a strong focus on **automation**, **home infrastructure**, and **hardware hacking**. I build tools that solve real problems — from intelligently tracking logistics pipelines to hacking Bluetooth speakers into smart home devices. I write primarily in Python, but reach for Rust, C++, and ESPHome when the job calls for it.

- Automating logistics tracking pipelines with web scraping and email parsing
- Building Home Assistant integrations and IoT hardware mods
- Syncing, organising and managing self-hosted photo infrastructure
- Experimenting with embedded systems (Arduino, ESP32) and wireless communication
